

# Online Examination System
*A simple Java console project for online test simulation.*

---
##  Overview
The **Online Examination System** is a Java console application that lets users take an online test.  
Users can log in, update their profile, select answers for questions, and their test will be auto-submitted after a timer ends.  
This project helps understand basic **Java programming concepts** like classes, objects, collections, and timers.

---

## Features
-  Login using username and password  
-  Update your profile details  
-  Select answers (A/B/C/D) for any question  
-  60-second timer with auto-submission  
-  View all your saved answers before logout  
-  Manual submit and logout anytime  

---

##  Concepts Used
- **OOP (Classes & Objects)**  
- **HashMap** to store answers  
- **Timer & TimerTask** for countdown and auto-submit  
- **Scanner** for user input  

---

##  How to Run
1. Open `OnlineExamination.java` in your Java IDE (IntelliJ, Eclipse, or VS Code).  
2. Compile and run the program.  
3. Follow the on-screen menu:  
Login
Update Profile
Select Answer
Submit & Logout
Exit


---

## Example Flow
1. Log in with username and password.  
2. Timer starts for 60 seconds.  
3. Select answers for questions.  
4. When the time ends, your answers are automatically submitted.  
5. The program logs you out and shows your answers.
